# Shoe Store Demo

Small vue3 / Vite/ Vuetify deamo using data from a websocket

##### Installation

Clone Repo

```
cd shoe-store
cd vue-frontend
npm install
```

##### Run the Websocket

```
cd shoe-store
websocketd --port=8080 ruby inventory.rb
```

##### Run the App

```
cd vue-frontend
npm run dev
```

visit http://localhost:5173
